# Multi-Agent SAP Enterprise Automation System

## Business Problem

Enterprise operations are distributed across multiple systems such as SAP, email, and incident management platforms, requiring manual coordination to execute tasks.

## Business Flow

User submits request → LLM understands intent → system decides required actions → MCP orchestrates across systems → results are executed and returned

## Solution

Designed a multi-agent system using Model Context Protocol (MCP), n8n, and Generative AI to dynamically orchestrate enterprise workflows across SAP and external systems.

## Impact

* Reduced manual effort
* Improved execution speed
* Enabled intelligent automation across systems

## Tech Stack

SAP CPI, MCP, n8n, LLM, RAG, Vector DB
